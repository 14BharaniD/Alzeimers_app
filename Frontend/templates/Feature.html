Project Overview

This project presents a novel multimodal framework for the early detection of Alzheimer's Disease (AD) by analyzing both handwriting and speech patterns. The system leverages two complementary approaches to provide accurate, non-invasive screening.

Handwriting Analysis

Analyzes handwriting samples for micrographia, tremors, and spatial disorganization - early markers of cognitive decline.

Speech Analysis

Processes speech recordings to detect hesitations, word-finding difficulties, and syntactic simplifications.

AI Integration

Combines machine learning (Random Forest) and deep learning (CNN) models for improved diagnostic accuracy.

Clinical Application

Provides a non-invasive, cost-effective screening tool for early Alzheimer's detection in clinical settings.

Key Objectives

  • Early detection of Alzheimer's disease 6-12 months before clinical diagnosis
  • Non-invasive screening using natural behavioral markers
  • Development of explainable AI models for clinical trust
  • Creation of a scalable, cost-effective diagnostic tool

Dataset Overview

The framework utilizes two comprehensive datasets for multimodal analysis of Alzheimer's disease biomarkers:

Handwriting Dataset

Contains 2,500 handwriting samples from 450 participants, including drawings, signatures, and written text tasks.

  • Healthy Controls: 180 participants
  • Mild Cognitive Impairment: 150 participants
  • Alzheimer's Patients: 120 participants

Speech Dataset

Includes 1,800 audio recordings from the same participant pool performing verbal cognitive tasks.

  • Picture description tasks
  • Story recall exercises
  • Verbal fluency tests
  • Sentence repetition tasks

Clinical Validation

All samples are clinically validated by neurologists and labeled according to standardized diagnostic criteria (NIA-AA).

Data Balancing

Applied SMOTE and data augmentation techniques to address class imbalance and improve model generalization.

Dataset Statistics

Total Participants: 450
Total Samples: 4,300

Methodology & Preprocessing

The data undergoes extensive preprocessing and feature extraction before model training:

1

Data Acquisition

Collect handwriting samples using digitizing tablets and audio recordings in controlled acoustic environments.

2

Handwriting Preprocessing

Normalize samples, remove artifacts, extract kinematic features (velocity, acceleration, pressure), and compute spatial features.

3

Speech Preprocessing

Apply noise reduction, voice activity detection, normalization, and convert to mel-spectrograms for CNN input.

4

Feature Engineering

Extract 85 handwriting features and 120 speech features, then apply PCA for dimensionality reduction.

5

Model Training

Train Random Forest on handwriting features and CNN on speech spectrograms using 10-fold cross-validation.

Technical Implementation

Python Libraries

Hardware

NVIDIA RTX 3080 GPU, 32GB RAM, Intel i9 processor for model training and inference

Model Architecture

Random Forest: 100 trees, max depth 20. CNN: 5 convolutional layers, 3 dense layers, dropout 0.5

Training Parameters

Batch size: 32, Epochs: 100, Learning rate: 0.001, Optimizer: Adam, Loss: Categorical Crossentropy

Prediction System

The deployed system allows for real-time analysis of new samples through an intuitive pipeline:

1

Sample Submission

Users upload handwriting images (PNG/JPG) or audio recordings (WAV/MP3) through the web interface or mobile app.

2

Automated Processing

System automatically applies the appropriate preprocessing pipeline based on input type.

3

Feature Extraction

Real-time extraction of relevant features using optimized algorithms.

4

Model Inference

Features are fed to the trained models (Random Forest for handwriting, CNN for speech).

5

Result Delivery

System provides prediction with confidence score, visual explanations, and comparative analysis.
